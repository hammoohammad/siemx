# SIEMX Dashboard (Frontend)

A modern, dark-mode SIEM dashboard built with React + Vite, Tailwind CSS, Recharts, and lucide-react.

## Tech
- React 18 + Vite
- Tailwind CSS 3
- React Router 6
- Recharts
- lucide-react icons

## Scripts
- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run preview` — preview production build

## Setup
1. Install dependencies
   ```bash
   npm i
   ```

2. Install vite 
   ```bash
   npm install vite
   ```

3. Start dev server
   ```bash
   npm run dev
   ```

## Structure
- `src/components` — shared UI
- `src/screens` — route pages
- `src/data/mock.js` — mock data + realtime simulation
